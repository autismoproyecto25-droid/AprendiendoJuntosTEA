Recursos educativos para TEA

Materiales organizados por objetivo para que familias y docentes puedan elegir rápidamente qué trabajar. Los apoyos visuales y de comunicación tienen buen respaldo científico como herramientas educativas complementarias.

PI

Pictogramas básicos

Tarjetas para pedir ayuda, agua, descanso, baño, comida y objetos de uso diario.

RV

Rutina de mañana

Secuencia visual para despertar, asearse, vestirse, desayunar y preparar mochila.

EM

Semáforo emocional

Guía visual (no clínica) para reconocer calma, alerta y saturación con acciones de regulación.

FR

Frases funcionales

Modelos breves para practicar peticiones, rechazo respetuoso y turnos de conversación.

TR

Tabla de transiciones

Apoyo para anticipar cambios entre juego, comida, escuela, descanso y salida.

CA

Kit de calma

Lista de objetos y acciones para crear un espacio de regulación sensorial en casa.

Cómo usarlos

Aplicación sencilla en tres pasos

1

Elige una meta

Trabaja una habilidad por vez: pedir ayuda, esperar turno o terminar una rutina.

2

Usa apoyos visibles

Coloca el material donde se necesita: mesa, baño, habitación o mochila.

3

Repite con calma

La práctica breve y constante suele funcionar mejor que sesiones largas.

Fuentes científicas

  • PubMed — Literatura científica revisada por pares sobre comunicación aumentativa y alternativa (se abre en una pestaña nueva) Base de datos donde se indexan las revisiones sistemáticas sobre pictogramas y sistemas de comunicación.
